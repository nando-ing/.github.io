Abstract

Measure Theory is an area devoted to study the size of sets and what set is "relevant" for certain operations.

Measure Theory

Measure Theory was created as an attempt generalise the size of a set. For example:
  • An interval $(a,b) \subset \mathbb{R}$ has size $(b-a)$
  • A rectangle $(a_1,b_1) \times (a_2, b_2) \subset \mathbb{R}^2$ has size $(b_1 - a_1)(b_2 - a_2)$
  • A n-dimensional rectangle $(a_1,b_1) \times \cdots \times (a_n, b_n) \subset \mathbb{R}^n$ has size $(b_1 - a_1) \hdots (b_n - a_n)$
As one can see, the concept of size is a generalisation of "volume". The sets in the example are all trivial ones, this concept is used to deal with highly complex set cases, for example, the Cantor set or the Image of a Dirichlet function. We can already note that the main objective is to obtain a function $\mu: \mathcal{F} \to \mathbb{R} \cup \{+\infty\}$, where $\mathcal{F}$ is a collection of sets, where $\mu$ should have the same desirable properties of the classical size of a set, as we can see in Definition \ref{def:measure_function}.
